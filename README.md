# MetaRefund Landing Page Deployment Guide

Эта страница описывает, как выгрузить одностраничный сайт MetaRefund в открытый доступ так, чтобы им можно было поделиться ссылкой и открыть в Google Chrome.

## 1. Подготовьте репозиторий GitHub
1. Создайте бесплатный аккаунт на [github.com](https://github.com), если его ещё нет.
2. Авторизуйтесь и нажмите **New repository**.
3. Задайте имя репозитория (например, `meta-refund-site`) и оставьте его публичным.
4. Нажмите **Create repository**.

## 2. Загрузите файлы сайта
1. На странице репозитория нажмите кнопку **Add file → Upload files**.
2. Перетащите в браузер файлы `index.html`, `styles.css`, `script.js` и папку `MetaRefund`, если она содержит картинки/иконки.
3. Внизу страницы добавьте комментарий к коммиту и нажмите **Commit changes**.

## 3. Включите GitHub Pages
1. Перейдите во вкладку **Settings** репозитория.
2. В боковом меню выберите **Pages**.
3. В секции **Build and deployment** задайте:
   - **Source**: `Deploy from a branch`.
   - **Branch**: `main` и папку `/ (root)`.
4. Нажмите **Save**. Через 1–2 минуты GitHub соберёт сайт и выдаст ссылку вида `https://<ваш-логин>.github.io/<имя-репозитория>/`.

## 4. Откройте сайт в Chrome
1. Скопируйте опубликованную ссылку из блока **GitHub Pages**.
2. Откройте Google Chrome и вставьте ссылку в адресную строку.
3. Готово — сайт доступен публично и работает в Chrome.

## 5. Альтернативный способ: Vercel/Netlify (по желанию)
Если нужен более гибкий хостинг:
1. Зарегистрируйтесь на [vercel.com](https://vercel.com) или [netlify.com](https://www.netlify.com/).
2. Подключите ваш GitHub-репозиторий.
3. Выберите проект и подтвердите деплой. Статический сайт будет доступен по новой ссылке.

## 6. Быстрый локальный запуск для проверки
Чтобы проверить сайт перед публикацией:
1. Установите [Node.js](https://nodejs.org/).
2. В корне проекта выполните в терминале команду `npx serve`.
3. Откройте в Chrome адрес `http://localhost:3000` (или порт из вывода команды).

Теперь у вас есть инструкции, как сделать сайт общедоступным и открыть его в Chrome.
